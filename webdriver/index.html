Learn how to use WebDriver

WebDriver is an emerging W3C standard through which web developers can write tests to automate testing their sites through web browsers. This demo page will show you how to use a number of WebDriver commands implemented in Microsoft Edge. The full list of commands implemented in Microsoft Edge can be found here.

Before trying the demo, please make sure you have installed Microsoft WebDriver server from Microsoft Download Center. After it is installed, start up the Microsoft WebDriver server by running the executable. Since you will be driving Microsoft Edge, we recommend running this demo in another browser. You're set to start sending some commands!

Command

The command to be sent to Microsoft Edge and control the browser. Below you'll find a dropdown of a few sample commands to play around with.

Method

The HTTP method used for each command. Every command will either be a GET, POST or DELETE.

Path

The path of the command selected. This is used by Microsoft WebDriver to determine the command. Any commands not in the supported commands list will get a "command not implemented" error response.

Request Body

This is the body of the request being sent to the Microsoft WebDriver server. Some commands will have values that you can add here, others do not require a request body.
