Learn how WebDriver works

Welcome to the Microsoft Edge Microsoft WebDriver demo! This demo will give you an idea of how WebDriver works in Microsoft Edge. Below you can select from a subset of commands that we have implemented in Microsoft Edge. For the full list you can visit our webdriver page. To begin, you'll first need to download and install Microsoft WebDriver from Download Center. After it is installed, start up the server by running the executable located in the folder you installed it into. Next open this page in Internet Explorer (since we will be driving Microsoft Edge, we can't run the demo from it) and start sending some comamnds!

Send Request

  • Command

    The command to be sent to Microsoft Edge. These allow control of the browser. Below you'll find a dropdown of a few sample commands to play around with.

  • Method

    The HTTP method used for each command. Every command will either be a GET, POST or DELETE.

  • Path

    The path of the command selected. This is used by Microsoft WebDriver to determine the command. By default Microsoft WebDriver has a list of approved commands. Any commands not in that list get a generic error response.

  • Request Body

    This is the body of the request being sent to the Microsoft WebDriver server. Some commands will have values that you can add here, other do not require a request body.
