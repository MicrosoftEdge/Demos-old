Professional Quality Video Updated - Interoperable Playback

About this demo:

Play DRM protected media using Encrypted Media Extensions (EME). Adjust the bitrate slider to simulate adaptive streaming by changing the video quality midstream. Play captioned media using built-in TTML captioning.

What's new:

Demonstrate playback using different EME API versions and different DRM provider support across multiple browsers. Automatically use the updated APIs in Microsoft Edge while still supporting vendor-prefixed EME APIs in IE11. Play DRM protected content identified as "PlayReady/Widevine" in Internet Explorer 11, Microsoft Edge and Google Chrome.

Featuring content hosting and DRM license acquisition services by Windows Azure Media Services.
