Pasting Local Images

This demo shows a browser's ability to paste images from the local file system into a web site.
With modern browsers, local images that are referenced on the clipboard are made available during paste. With no script, images are pasted with dataURI. Switching to blob below runs simple javascript to determine the type of image object to use. See this MSDN reference for more details. Try pasting an image from PrintScreen, Microsoft Word, Paint, or other image software. Then check out the markup you've just pasted.
