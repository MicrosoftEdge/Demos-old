Web Note is a new feature of Microsoft Edge. It helps you write, draw and make notes directly on the web page. You can also share the web page with your notes on it.

This holiday themed coloring book will take you through how to use Web Note. Click the and scroll down to start with the monkey maze.

Click the browser's Web Note icon and then scroll to get started!

Complete the Maze

Click the Pen .

Trace a path through the maze to deliver the presents.

Scroll down to continue with Connect the Dots.

Connect the Dots

Select the pen tool again to select color and size. Then try connecting the dots to complete the Christmas tree.

You can clip a selection of the page to your clipboard by using the Clip tool . Once copied to the clipboard, you can paste it anywhere.

Select the pen or highlighter to exit Clip mode.

Scroll down to continue with the Holiday Word Jumble.

Holiday Word Jumble

Select a pen color. We recommend the medium size pen for this activity.

Draw a continuous line in any direction across the letters to cross out the holiday words hidden in the word jumble.

You can save your web notes and come back to the later by clicking Save and then choosing OneNote, Favorites or Reading List. Your Web Note will be saved and be available for you via OneNote, Favorites or Reading List.

Scroll down to continue with the coloring book.

Color in the Reindeer

Select the highlighter for coloring the reindeer so you can see the outline while you color.

Select a color and brush size to begin.

You can change your highlighter color and size at any time to do different parts of the reindeer.

You can also erase your markings by selecting the Eraser and then strike through your markups for them to be erased.

Switch back to the pen or highlighter when you’re done erasing.

Color in the Bells

Select the highlighter for coloring the bells so you can see the outline while you color.

Select a color and brush size to begin. You can change your highlighter color and size at any time to do different parts of the bells.

Color in the Wreath

Select the highlighter for coloring the wreath so you can see the outline while you color.

You can clip a selection of the page to your clipboard by using the Clip tool .

You can also email your clip or share it on Facebook or Twitter via Share .

You can also have Facebook or Twitter show as options on the Share panel once you install the apps from Windows Store.

Scroll down for more holiday fun to color.

Color in the Snowflakes

Select the highlighter for coloring the snowflakes so you can see the outline while you color.

If you open Web Note on a web page that has content to the right or left of what you can see, you can pan the web note view by using the Pan tool .

Touch one of the other Web Note tools to exit panning mode.

Scroll down for more holiday fun to color.

Color in the Snowmen

Select the highlighter for coloring the snowmen so you can see the outline while you color.

You can save your Web Note and come back to it later by clicking Save and then choosing OneNote, Favorites or Reading List. You can access your Favorites or Reading list via Hub . You can open your OneNote app from the Start menu.

Scroll down to continue with the coloring book. You can exit Web Note mode at any time by clicking Exit. If you haven’t saved your Web Note, it will be discarded when you exit. So be sure to Save if you want to continue again later.

Scroll down for the final picture to color.

Color in the Sleigh

Select the highlighter for coloring the sleigh so you can see the outline while you color.

You can add a typed comment to the page by selecting the Typed note icon . Use your keyboard to type a note and use your mouse to drag the position of the note around the page and collapse the note.

You can expand and collapse notes using the note anchor on the page.

You can exit Web Note mode by clicking Exit. Any unsaved notes you’ve made will be discarded and you’ll return to the web page.
