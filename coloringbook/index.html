The Web Note coloring book and practice guide

Web Note is a new feature of Microsoft Edge. It helps you write, draw and make notes directly on the web page. You can also share the web page with your notes on it.

This autumn themed coloring book will take you through how to use Web Note. Click the Web Note icon and scroll down to start with the maze.

Click the browser's Web Note icon and then scroll to get started!

Complete the maze

  1. Click the Pen icon.
  2. Trace a path through the maze to get to the acorn.
  3. Scroll down to continue with connect the dots.

Connect the dots

  1. Select the pen tool again to select color and size. Then try connecting the dots to complete the autumn leaves.
  2. You can clip a selection of the page to your clipboard by using the Clip tool. Once copied to the clipboard, you can paste it anywhere.
  3. Select the pen or highlighter to exit Clip mode.
  4. Scroll down to continue with the autumn word jumble.

Autumn word jumble

  1. Select a pen color. We recommend the medium size pen for this activity.
  2. Draw a continuous line in any direction across the letters to cross out the autumn words hidden in the word jumble.
  3. You can save your web notes and come back to the later by clicking Save and then choosing OneNote, Favorites or Reading List. Your Web Note will be saved and be available for you via OneNote, Favorites or Reading List.
  4. Scroll down to continue with the coloring book.

Color in the cornucopia

  1. Select the highlighter for coloring the cornucopia so you can see the outline while you color.
  2. Select a color and brush size to begin.
  3. You can change your highlighter color and size at any time to do different parts of the cornucopia.
  4. You can also erase your markings by selecting the Eraser and then strike through your markups for them to be erased.

Color in the turkey

  1. Select the highlighter for coloring the turkey so you can see the outline while you color.
  2. You can clip a selection of the page to your clipboard by using the Clip tool.

  5. Scroll down for more autumn fun to color.

Color in the feast

  1. Select the highlighter for coloring the feast so you can see the outline while you color.
  2. If you open Web Note on a web page that has content to the right or left of what you can see, you can pan the web note view by using the Pan tool.

  3. Touch one of the other Web Note tools to exit panning mode.
  4. Scroll down for more autumn fun to color.
