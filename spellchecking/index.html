Spellchecking in IE10

IE10 now supports HTML5 spellchecking. As you type text into textarea and contenteditable regions, the browser automatically checks your spelling.

IE10 Spellchecking brings the "best in class" spelling engine and dictionaries used by Microsoft to the browser. IE10 is also the first browser to support auto-correction of text while you type! Try it out.

The spellcheck attribute

Spellchecking is active by default on <textarea> and contenteditable elements, and off by default for text boxes

Web developers can override the defaults using the spellcheck attribute.

spellcheck=false (disabled)
spellcheck=true (enabled)
